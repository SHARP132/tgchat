# TGChat — Telegram-like Messenger

Полноценный чат-мессенджер в стиле Telegram. Работает в локальной сети или в интернете.

## Функции
- 💬 Личные сообщения (DM)
- 📢 Публичные каналы (создание и вступление)
- 👤 Профиль: аватар, ник, юзернейм, описание
- 📎 Отправка файлов и фото
- 🎤 Голосовые сообщения (удержание кнопки)
- 📹 Видеосообщения-кружки
- 😂 Реакции на сообщения
- ↩️ Ответы на сообщения
- 🌙 Тёмная тема и 4 других темы + кастомные цвета
- 🟢 Статус онлайн
- ✏️ Индикатор "печатает..."

## Запуск локально

```bash
npm install
npm start
```
Открой http://localhost:3000 в браузере.

## Деплой на Railway (бесплатно, в интернете)

1. Создай аккаунт на [railway.app](https://railway.app)
2. Залей код на GitHub
3. В Railway: **New Project → Deploy from GitHub repo**
4. Railway автоматически задеплоит — получишь публичный URL

## Деплой на Render (бесплатно)

1. Создай аккаунт на [render.com](https://render.com)
2. **New → Web Service → Connect GitHub**
3. Build command: `npm install`
4. Start command: `node server.js`

## Переменные окружения
- `PORT` — порт сервера (по умолчанию 3000)

## Структура
```
tgchat/
├── server.js       # Node.js + Express + Socket.io сервер
├── public/
│   ├── index.html  # UI
│   ├── style.css   # Стили (темы, компоненты)
│   └── app.js      # Клиентская логика
└── uploads/        # Загруженные файлы (создаётся автоматически)
```
